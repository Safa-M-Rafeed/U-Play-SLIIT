# User Management Module

## Components
- UserProfile.jsx      - profile update feature
- UserSearch.jsx       - search and filter
- UserPagination.jsx   - pagination for user list
- UserDashboard.jsx    - stats dashboard UI

## Backend
- session.js                  - session bug fix middleware
- loginRedirect.js            - login redirect after password reset
- userDeleteController.js     - delete and deactivate users
- AuditLog.js                 - audit log model
- userRegistration.test.js    - unit tests for registration
